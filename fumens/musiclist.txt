*loadalltxt

//===========================以下、説明========================
// このファイルは、選曲画面モード開始時に一度だけ読み込まれます。
// どのファイルを譜面として認識し、選曲画面上で選べるようにする
//  (以下では「リストに加える」と表現します) か、を
// このファイルで指定することができます。
// 
// ●ファイル名が具体的に
// evans_ext_memo.txt
// diamonddust.txt
// sayonara.txt
// 新しいフォルダー/cynthia.txt
// などと羅列されている場合は、それらのファイルのみをリストに加えます。
// (パスの区切りは \ でなく / を使ってください)
// 
// ● *loadalltxt と書いてあると（初期設定状態）、fumen フォルダにある全ての .txt ファイル
// (musiclist.txt は除く) をリストに加えます。
// (musiclist.txt がそもそも存在しない場合も、↑と同様の動作をします。)
// (*loadalltxt の手前にファイル名がいくつか羅列してある場合は、それらもリストに加えられます。)
// (*loadalltxt の後にファイル名がいくつか羅列してあっても、それらは無視されます。)
// 
// ●いずれの場合も、最初の行に *omiterror:n と書く (ただし n は正の整数) ことにより、
// (以下で述べるような)「エラー」の存在するファイルはリストに加えない、という指定ができます。
// 
// 4: 音楽ファイル名 (「m=」で指定する) が指定されていない
// 16: レベル (「#lev=」で指定する) が指定されていない
// 32: 曲名 (「#title=」で指定する) が指定されていない
// 64: アーティスト名 (「#artist=」で指定する) が指定されていない
// 128: ジャケットファイル名 (「#jacket=」で指定する) が指定されていない
// 1024: 選択時の音楽プレビュー開始位置 (「#prevpos=」で指定する) が指定されていない
// 
// 複数のエラーのうち 1 つ以上をもつファイルはリストに加えたくない、という場合は、
// 数値を合計して指定してください (たとえば *omiterror:20 と書けば、音楽ファイル名が
// 指定されていないファイルも、レベルが指定されていないファイルも、リストに加わりません)。
// 
// ●あと、最後の行にも改行をお忘れなく (→)
